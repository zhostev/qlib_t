# 前后端联调测试计划

## 项目结构概述
- **后端**: FastAPI 框架，运行在 8000 端口
- **前端**: React + TypeScript + Vite，运行在 3001 端口，配置了 /api 代理到后端

## 联调测试步骤

### 1. 安装依赖
- **后端**: 安装 requirements.txt 中的所有依赖
- **前端**: 确保 node_modules 已安装，如未安装则执行 npm install

### 2. 启动后端服务
- 进入 backend 目录
- 使用 uvicorn 启动 FastAPI 应用：`uvicorn main:app --reload --port 8000`
- 验证后端服务是否正常运行：访问 http://localhost:8000

### 3. 启动前端服务
- 进入 frontend 目录
- 使用 npm 启动开发服务器：`npm run dev`
- 验证前端服务是否正常运行：访问 http://localhost:3001

### 4. 测试前后端交互
- 测试 API 连接：前端是否能成功调用后端 API
- 测试数据交互：创建、读取、更新、删除操作是否正常
- 测试认证流程：如果有认证功能，测试登录登出
- 测试核心功能：如实验创建、运行、结果查看等

### 5. 调试和问题解决
- 检查浏览器控制台是否有错误
- 检查后端日志是否有错误
- 检查网络请求是否成功
- 必要时使用调试工具进行问题定位

## 预期结果
- 前后端服务都能正常启动
- 前端能成功调用后端 API
- 核心功能能正常工作
- 没有明显的错误或异常

## 注意事项
- 确保端口 8000 和 3001 没有被占用
- 确保数据库连接正常（如果后端使用了数据库）
- 确保环境变量配置正确
- 注意查看日志输出，及时发现问题
