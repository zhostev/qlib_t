# 联调测试与问题修复计划

## 问题分析
1. **后端数据库错误**：`factors`表缺少`group_id`字段，导致查询失败
2. **前端代理连接问题**：前端无法连接到后端服务，可能是后端服务不稳定
3. **数据库初始化问题**：虽然初始化脚本包含了处理逻辑，但可能没有正确执行

## 测试与修复步骤

### 1. 数据库修复
- 停止当前后端服务
- 运行数据库重置脚本：`python reset_db.py`
- 重新运行数据库初始化脚本：`python init_db.py`
- 验证数据库表结构是否正确

### 2. 后端服务测试
- 启动后端服务：`uvicorn main:app --host 0.0.0.0 --port 8000 --reload`
- 测试API端点：
  - GET /api/factors/groups
  - GET /api/factors/
  - GET /api/configs/
  - GET /api/auth/users
- 确保所有API端点都能正常响应

### 3. 前端服务测试
- 检查前端代理配置是否正确
- 启动前端服务：`npm run dev`
- 访问前端页面：http://localhost:3001
- 测试前端功能：
  - 登录功能
  - 因子管理页面
  - 实验管理页面
  - 仪表盘页面
- 检查浏览器控制台是否有错误

### 4. 联调测试
- 测试前后端交互：
  - 创建因子组
  - 创建因子
  - 运行实验
  - 查看实验结果
- 确保所有功能都能正常工作

### 5. 问题修复
- 根据测试结果修复发现的问题
- 确保所有API端点都能正常响应
- 确保前端页面能正常显示和交互
- 确保前后端数据传输正确

## 预期结果
- 后端服务稳定运行，所有API端点返回正确结果
- 前端页面正常显示，无控制台错误
- 前后端交互正常，所有功能可用
- 系统能够正常进行因子管理和实验运行
